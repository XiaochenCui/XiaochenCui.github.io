我们首先定义合法字符集为{英文字母，[,], [.], [“], [ ]}

由于所有密文都是用同一密钥加密的，所以对于密文 c1, c2: result = c1 ^ c2 ^ x (x 为合法字符）

result = m1 ^ key ^ m2 ^ key ^ x (m1, m2, k 均为密文 c1, c2 以及密钥串 key 在同一位置的值）

排除意外碰撞，当且仅当 m1 == x 或 m2 == x 时，result 为合法字符

  • 合法字符集的范围可能不准确
