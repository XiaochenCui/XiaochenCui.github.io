{% asset_img 1.png 700 %} 我们可以从 Padding oracle attack 中找到伪造密文的思路： 设：$1 \oplus x = 5$, IV 中的对应字节为 $y$ 则：当 $y’ = y \oplus x$ 时，$y \oplus 1 \equiv y’ \oplus 5$ 所以：将 $y$ 更改为 $y’$ 可使 message 中的相应字节变更为 5
